# Project Checklist ✅

Use this checklist to ensure health and readiness of the website.

- [x] Install dependencies (`npm install`)
- [x] Lint/compile without errors (`tsc` handled by Vite)
- [x] Run tests (`npm run test`) and maintain coverage >70%
- [x] Document new components or features in `README.md` or `docs/`
- [x] Update constants (menu items, contact info) in `constants.ts` when content changes
- [x] Verify mobile/responsive layout on real device or emulator
- [x] Check production build (`npm run build`) and preview (`npm run preview`)
- [ ] Add deployment/CI configuration if publishing to GitHub Pages/Vercel/etc.
- [ ] Review accessibility (aria labels, color contrast)
- [ ] Add user-facing tests or visual snapshots if needed

Feel free to copy this list into issue templates or project boards.
